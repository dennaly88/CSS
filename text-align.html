Text-Align

La propiedad CSS establece la alineación horizontal del contenido de nivel en línea dentro de un elemento de bloque o cuadro de celda de tabla. Esto significa que funciona como pero en la dirección horizontal.text-align
La text-alignpropiedad se especifica de una de las siguientes maneras:

Usando los valores de palabra clave start, end, left, right, center, justify, justify-allo match-parent. Usando solo un valor, en cuyo caso el otro valor predeterminado es right. Usando un valor de palabra clave y un valor.

Valores

start
Lo mismo que leftsi la dirección fuera de izquierda a derecha y rightsi la dirección fuera de derecha a izquierda.
end
Lo mismo que rightsi la dirección fuera de izquierda a derecha y leftsi la dirección fuera de derecha a izquierda.
left
El contenido en línea se alinea con el borde izquierdo del cuadro de línea.
right
El contenido en línea se alinea con el borde derecho del cuadro de línea.
center
Los contenidos en línea se centran dentro del cuadro de línea.
justify
Los contenidos en línea están justificados. El texto debe estar espaciado para alinear sus bordes izquierdo y derecho con los bordes izquierdo y derecho del cuadro de línea, excepto la última línea.
justify-all Experimental
Igual que justify, pero también obliga a que se justifique la última línea.
match-parent
Similar a inherit, pero los valores starty endse calculan de acuerdo con los padres directiony se reemplazan por el valor lefto apropiado right.
Experimental
Cuando se aplica a una celda de tabla, especifica el carácter de alineación alrededor del cual se alineará el contenido de la celda.

div.a { text-align: center; }
div.b { text-align: left; }
div.c { text-align: right; }
div.c { text-align: justify; }
